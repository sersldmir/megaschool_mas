Система для проведения автоматизированных технических интервью с использованием Mistral AI и LangGraph.
Проект имитирует реального рекрутера, который адаптирует вопросы под уровень кандидата, следит за таймингом и генерирует детальный фидбэк.

Ключевые моменты:

Мультиагентная архитектура: Разделение ролей на Интервьюера (ведет диалог) и Ментора (скрытый наблюдатель, принимающий решения).

Умное планирование (Roadmap): Система анализирует стек кандидата в начале и строит очередь тем (Topic Queue). Темы переключаются автоматически по мере прохождения.

Скрытая рефлексия (Chain of Thought): В логах видно, как Ментор анализирует ответ и какую инструкцию дает Интервьюеру, а также мысли Интервьюера перед генерацией вопроса.

Структурированный фидбэк: В конце интервью генерируется отчет с грейдом, сильными/слабыми сторонами и рекомендациями по обучению.

Устойчивость (Robustness): Система защищена от ошибок форматирования JSON со стороны LLM (валидация через Pydantic + Defensive Coding).

Система построена как направленный граф состояний. Один ход системы (Turn) проходит следующую цепочку:

User Input: Кандидат отправляет сообщение.
Profile Parser (только в начале): Извлекает роль/грейд и генерирует список тем (topic_queue).
Mentor Node:
Оценивает ответ (correct / incorrect).
Генерирует скрытую инструкцию для Интервьюера.
Interviewer Node:
Получает инструкцию.
Генерирует естественный вопрос.
Формирует запись в лог (Мысли Ментора + Мысли Интервьюера + Вопрос).
Feedback Node (в конце): Генерирует итоговый отчет.
